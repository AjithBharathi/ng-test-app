inbuilt-pipes works!

✅ 1. uppercase

Converts a string to UPPERCASE.

{{ 'angular' | uppercase }}

✅ 2. lowercase

Converts a string to lowercase.

{{ 'ANGULAR' | lowercase }}

✅ 3. titlecase

Capitalizes the first letter of each word.

{{ 'angular is awesome' | titlecase }}

✅ 4. slice

Slices a string or array (like substring() or slice() in JS).

{{ 'Angular' | slice:1:4 }}

{{ [10, 20, 30, 40] | slice:1:3 }}

✅ 5. date

Formats a Date object into a readable string.

{{ today | date:'short' }}

{{ today | date:'fullDate' }}

✅ 6. currency

Formats a number into currency (₹, $, etc.).

{{ 1234.5 | currency:'INR':'symbol':'1.2-2' }}

{{ 1234.5 | currency:'USD' }}

✅ 7. percent

Displays a number as a percentage.

{{ 0.25 | percent }}

{{ 0.259 | percent:'1.1-2' }}

✅ 8. number

Formats a number with a specific number of decimals.

{{ 3.141592 | number:'1.1-2' }}

Format: 'minInt.minFraction-maxFraction'

✅ 9. json

Converts a JS object to a JSON string (great for debugging).

{{ { name: 'Ajith', age: 30 } | json }}

✅ 10. keyvalue

Transforms an object or Map into an array of key-value pairs for iteration.

{{ item.key }}: {{ item.value }}

✅ 11. async

Automatically subscribes to Observables or Promises.

{{ user$ | async }}

Hello {{ user.name }}

✅ 12. i18nPlural

Handles pluralization for different numbers.

{{ messageCount | i18nPlural:messages }}

✅ 13. i18nSelect

Displays a message based on a selection value (gender, role, etc.)

{{ userGender | i18nSelect:genderMap }}
